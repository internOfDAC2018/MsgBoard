# Board
一个ios应用 用于共享信息

用户发送的内容在 “留言板” 上显示。仿照了微博应用在显示微博动态的页面。

## 当前预计基础版本的功能：
### 用户：
* 登录 
* 注册 
### 消息：
* 留言只能附加一张图片 
* 可以评论
### 留言板：
* 分页 
* 查看详细 
* 按照消息发送者过滤消息

## 高级功能：
### 消息
* 可指定 “谁能看到”
